Python Tutorial

String
    .upper(): to change the words to upper case
    .lower(): to change the words to lower case
    .capitalize(): to capitalize the first letter of a sentence or a word
    .count(): to count a specific caracter
    .format: used to shorten a long code specially for concatination
        eg: output = f'Hello, {first_name} {last_name}'
            print(output)
    .replace(old,new): to replace sth in String
    len(string): to find the length of the string
    .sort(): to sort string alphabetically
    sorted(string): sort the string in alphabetical order
    .split(): split a string using a specific character and store it in a List
    eval(string): to change string to object or other types of things
